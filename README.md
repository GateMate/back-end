# Backend
Back End Code for GateMate


**Setting up and running flask app `server.py`**

*Install Flask*
`pip install flask`



*Mac setup*
```
export FLASK_APP=server.py
flask run
```
*Windows setup*
Command Prompt
```
set FLASK_APP=server.py
flask run
```
Power Shell
```
flask --app server.py run
```

**Using Firebase SDK**
```
pip install firebase
```
