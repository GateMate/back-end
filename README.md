**Setting up and running flask app `server.py`**

*Install Flask*
`pip install flask`



*Mac setup*
```
export FLASK_APP=server.py
flask run
```
*Potential Windows set up*?
Command Prompt
```
set FLASK_APP=server.py
flask run
```
Power Shell
```
env: FLASK_APP=server.py
flask run
```

**Using Firebase SDK**
```
pip install firebase
```
